# Human-vs-AI Evaluation Corpus — MANIFEST

Built: 2026-05-23. All texts are **real** human writing or **real published model
output**. No AI text was generated by us for this corpus.

## Contents

- `corpus/human/*.txt` — 70 human-authored texts
- `corpus/ai/*.txt` — 62 AI-generated texts
- `metrics.csv` — one row per text, engine metrics (see schema below)

Total: **132 texts** (70 human / 62 AI). Both classes ≥ 40 (minimums met).
Filenames encode provenance: `h_`/`a_` prefix = human/ai; then source tag.

## Sources & provenance (how they were fetched)

### 1. HC3 — Hello-SimpleAI/HC3 (`all.jsonl`)
- Fetched via `curl -sSL https://huggingface.co/datasets/Hello-SimpleAI/HC3/resolve/main/all.jsonl`
  (73.7 MB, HTTP 200).
- Each record has a real `human_answers` list and a `chatgpt_answers` list
  (ChatGPT, late 2022) for the same question, plus a `source` domain tag.
- We took matched human/AI **pairs** per domain:
  - `reddit_eli5` → domain **general** (12 pairs)
  - `open_qa` + `wiki_csai` → domain **qa** (12 pairs)
  - `finance` → domain **finance** (10 pairs)
  - `medicine` → domain **medicine** (10 pairs)
- License: HC3 is released under CC-BY-SA-4.0 (per dataset card / repo).

### 2. ChatGPT-Research-Abstracts — NicolaiSivesind/ChatGPT-Research-Abstracts
- Fetched via `curl -sSL .../resolve/main/research_abstracts-final.csv` (26.5 MB, HTTP 200).
- CSV columns: `title, real_abstract, real_word_count, generated_abstract, generated_word_count`.
- `real_abstract` = human scientific abstract; `generated_abstract` = produced by
  **GPT-3.5-turbo-0301** (prompted with the real title, matched target length).
- Used for the **scientific** register: 18 matched human/AI pairs.
- License: `cc` (per dataset card). Created by N.T. Sivesind & A.B. Winje (bachelor thesis).

### 3. arXiv API — extra clearly-human scientific text
- Fetched live with stdlib `urllib` from `http://export.arxiv.org/api/query`,
  restricted to `submittedDate:[2019..2020]` (pre-LLM → unambiguously human).
- Categories cs.LG, q-bio.NC, stat.ME; 8 abstracts kept.
- These are **human-only** additions (the 8-text human/AI imbalance comes from here).
- arXiv abstracts: metadata via arXiv API; abstracts are author-written, used here
  for non-commercial research evaluation only.

## Cleaning

- HTML-unescaped; stripped markdown markup, code ticks, links, URLs; collapsed
  whitespace; fixed HC3 token-spacing artifacts (` ?` → `?`).
- Kept only texts with **40 ≤ words ≤ 400**. Class word-count distributions:
  human mean ≈ 178 (42–400), ai mean ≈ 224 (83–377) — comparable but see caveats.

## metrics.csv schema

`id,label,domain,source,n_words,density,cv,opener_entropy,func_word_ratio,n_findings,low_conc_paras`

Produced by running `python3 plainly/scripts/prescan.py <file> --json` on each text
(`eval/scripts/run_metrics.py`, stdlib-only). `cv` is blank for 11 short texts where
burstiness is undefined (too few sentences) — this is genuine engine behavior.

## Results (engine signal)

Mean density: **human 0.260 vs AI 1.155** (≈4.4×). AI > human density in every
domain. AI also shows more findings (0.73 vs 0.14/text) and lower burstiness CV
(0.319 vs 0.527). The engine separates the classes without being trained on them.

## Caveats (read before trusting numbers)

- **Contamination / provenance of "AI":** HC3 ChatGPT answers are late-2022; the
  scientific AI side is GPT-3.5-turbo-0301 (March 2023). Newer models write
  differently, so absolute density gaps may not transfer to current LLMs.
- **Length confound:** AI texts average ~46 more words than human. Density is a
  per-text rate, but longer texts have more chances to trip findings. Treat the
  density gap as suggestive, not adjusted for length.
- **Register coverage is uneven:** scientific is the largest single register and is
  the only one with an extra human-only source (arXiv), so the human scientific set
  mixes two provenances (abstract dataset + arXiv API).
- **Domain labels** follow the source datasets' own tags; "qa" merges open_qa and
  wiki_csai.
- **Selection:** we took the first N qualifying records per domain (deterministic,
  not random) — fine for a smoke-test eval, not a statistically sampled benchmark.
- **Licensing:** HC3 (CC-BY-SA-4.0) and ChatGPT-Research-Abstracts (CC) permit
  research reuse; arXiv abstracts retained under fair research use. Redistribution
  of derived `.txt` should preserve these terms.
